## 1. 广告法极限词（绝对禁止）

以下词汇在商业推广、产品介绍、服务描述中一律禁用（非广告性客观叙述可酌情判断）：

### 最高级/排名类
```
最、第一、唯一、顶级、极致、终极、最强、最优、最佳、最大、最小
国内首家、全国第一、行业领先、遥遥领先、业界标杆
```

### 绝对化/保证类
```
绝对、肯定、必定、保证、确保、永久、永远、永不、终身
100%、百分百、万无一失、万全
```

### 权威背书类（未经授权）
```
国家级、国家认定、国际认证、获国家认可
央视推荐、政府指定、官方认证（未经授权不可用）
```

## 2. 医疗健康类禁用词

```
治愈、根治、彻底根除、永久消除
包治百病、对症下药（非医疗机构）
纯天然无副作用（涉及药品/器械）
临床验证（无资质时）
专家推荐、医生建议（无具体信息时）
抗癌、防癌、降血糖（保健品类）
```

---

## 3. 食品保健品禁用词

```
治疗（食品无法声称治疗疾病）
减肥效果立竿见影
快速增高/增重
排毒养颜（涉及疾病预防暗示）
改善记忆力（无科学依据时）
增强免疫力（无获批功能时）
```

---

## 4. 金融理财类禁用词

```
保本保息、稳赚不赔
高收益（无风险警示时）
年化 XX%（须同时展示风险警示）
无风险、低风险（涉及投资时）
抄底、必涨、必跌（证券投资类）
```

## 5. 诱导关注/分享/转发（微信平台明确禁止）

### 禁止的诱导行为
- **诱导分享朋友圈**："分享到朋友圈可领取……""转发给 X 个好友获得……"
- **诱导关注**："关注后回复 XX 可获得……"（若关注后无实质服务，属诱导）
- **诱导点击**："点击阅读全文即可领取……"
- **诱导传播**："帮我们转发 10 次，我们就……"
- **威胁性诱导**："不转发的人都是……"

### 微信平台处理方式
- 触发诱导分享检测：文章可能被限制传播、降低推荐权重
- 多次违规：账号被降权、封禁群发功能

## 6. 其他营销违规行为

### 虚假促销
- 虚构原价（"原价 9999 元，现在仅需 99 元"且原价未实际销售过）
- 虚假限时（"仅剩最后 3 个名额"但长期保持此提示）
- 虚假好评（展示刷出的用户评价）

### 违规抽奖/赠品
- 组织抽奖须符合《彩票管理条例》，不得设置强制消费门槛
- "转发抽奖"属于诱导分享，明确违规
- 不得以"恭喜您中奖"诱骗用户点击链接

### 会员/付费内容
- 不得虚构会员特权
- 不得设置自动续费而未明显告知用户

## 7. 互动引导的合规写法

合规的互动引导应基于内容价值，而非奖励驱动：

| 违规写法 | 合规替代写法 |
|---------|------------|
| 转发本文可领取礼品 | （删除，内容自然传播）|
| 关注后回复"666"抽奖 | 关注后可获取我们的内容更新 |
| 点个赞再走吧！ | （正常使用，无奖励承诺即可）|
| 帮我们转发 10 次 | （删除）|
| 分享给朋友赚红包 | （删除，等同于传销）|

## 8. 常见合规 vs 违规对照

| 场景 | 违规写法 | 合规写法 |
|------|---------|---------|
| 产品推广 | "全国最好的 XX 产品" | "我们的 XX 产品获得了 XX 用户好评" |
| 价格促销 | "史上最低价" | "本月特惠，低至 XX 元" |
| 健康产品 | "彻底根治 XX 病" | "有助于改善 XX 状况（非医疗建议）" |
| 投资产品 | "年化 20%，稳赚不赔" | "目标收益 20%，投资有风险，请谨慎决策" |
| 互动引导 | "转发抽 iPhone" | （无奖励互动引导，如：欢迎评论分享您的看法）|
| 权威背书 | "国家认证，权威推荐" | 展示具体认证证书名称和编号 |
