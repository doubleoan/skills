# 标题合规规则

> 规则来源：微信公众平台运营规范、《互联网新闻信息服务管理规定》、平台公告、行业实践总结

---

## 1. 字数与格式要求

- **推荐字数**：15–30 字（最佳传播区间）
- **最大字数**：64 字（超出截断显示）
- **禁止全部大写字母**（中文无此问题，但纯英文全大写标题有风险）
- **标点符号**：避免堆叠（如连续 `！！！`、`???` 等）
- **表情符号**：可使用，但不得替代正常文字表达意思

## 2. 禁止的标题类型

### 2.1 标题党 / 夸大虚假
- 使用"震惊""不得了""你不知道的秘密"等夸大词
- 与正文内容严重不符（标题党）
- 故意制造悬念但正文无实质内容
- 使用未经证实的数字或比例（"90% 的人都不知道"）

### 2.2 谣言与虚假信息
- 发布未经证实的重大新闻
- 传播已被官方辟谣的内容
- 假借权威机构名义发布虚假信息（"央视曝光""新华社独家"）

### 2.3 政治敏感
- 涉及领导人、党政机关负面表述
- 涉及民族、宗教、地区歧视性表达
- 涉及台湾、西藏、新疆等敏感地区独立性表述
- 对政策措施进行恶意解读或攻击性评论

### 2.4 诱导行为
- "转发本文可获得……"
- "分享到朋友圈即可领取……"
- "点击关注即可……"（诱导关注）

### 2.5 低俗与擦边球
- 涉及色情、性暗示的标题
- 使用低俗词汇博取眼球
- 以"福利""资源""懂的都懂"等词暗示违规内容

## 3. 高风险标题特征（极易触发审核）

| 特征 | 示例 | 风险等级 |
|------|------|---------|
| 使用极限词 | "最强、最权威、最全" | 中 |
| 虚假数字 | "99% 的人都……" | 中-高 |
| 煽动情绪 | "愤怒了！中国人必须……" | 高 |
| 假冒权威 | "官方通知""紧急公告" | 高 |
| 政治敏感 | 涉及领导人姓名+负面词 | 极高 |
| 涉及死亡/灾难 | "××人已死" "重大事故" | 高 |
| 诱导分享 | "转发后……" | 高 |
| 低俗擦边 | 性暗示词汇 | 高 |

---

## 4. 低风险标题特征

- 标题与正文内容高度一致
- 有具体信息量（时间、地点、人物、事件）
- 不使用极限词，表达客观
- 不依赖悬念吸引点击
- 字数适中（15–30 字）

---

## 5. 常见违规模式示例

**违规标题 → 改写建议：**

| 违规标题 | 问题 | 改写建议 |
|---------|------|---------|
| 震惊！这件事你一定不知道 | 标题党，无实质信息 | 直接说出"这件事"是什么 |
| 100% 有效的减肥方法！ | 极限词 + 夸大 | "这 3 个减肥习惯，坚持 1 个月有什么变化" |
| 转发本文，抽取 iPhone | 诱导分享 | 删除奖励描述，内容自然传播 |
| 紧急！央视曝光黑心商家 | 假冒权威 + 煽情 | "这类商家被央视曝光，消费者要注意" |
| 懂的都懂，这个视频不简单 | 暗示违规内容 | 明确说明视频内容主题 |
