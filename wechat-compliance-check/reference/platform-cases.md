# 违规案例解析

> 来源：微信公众平台官方文档、微信规则中心（https://mp.weixin.qq.com/webpoc/ruleCenter）
> 用途：作为合规检查的具体案例参考，每条案例对应具体的违规情形和处罚结果

---

## 一、低创作度内容

平台建议创作者产出信息含量高或能提供其他阅读价值的内容，持续、大量发布低创作度（如高度同质化、搬运抄袭、内容空洞、低质AIGC）的作品，不仅难以为用户提供有价值信息，也将直接影响创作者在平台所获得的推荐机会。

### 1.1 同质化内容

短期内，频繁发布主题、标题、正文、封面高度相似的内容，缺乏信息增量，容易让用户产生重复阅读的困扰。常常表现为账号内容整体重复率过高。

**案例1：** 重复发表——标题、封面、正文均完全一致，且在短期内重复发表。

**案例2：** 框架一致——所发内容结构完全一致，均套用"剧名+xx致死不知+而是"的格式以及内容框架，且在短期内大量发表。

### 1.2 疑似抄袭、搬运内容

大篇幅复述站内、外的内容，或通过机械拼接、洗稿重组等方式进行轻加工，整体内容不含个人观点、分析等增量信息、创作度低。

**案例1：** 直接对站外内容进行截图搬运，不含自我观点等进一步表达，属于典型的直接搬运内容。

### 1.3 低信息量内容

信息量不足的内容无法为用户提供阅读价值，包括简单拼凑无意义信息、图文无关或看图说话、引用存疑的数据/案例/言论为主题、或无依据情况下进行效果演示/夸大等各种类型的内容。

**案例1：** 简单拼凑的低信息量、无意义内容，无阅读价值。

### 1.4 低价值AIGC内容

平台不反对创作者使用新技术进行辅助创作，例如提供参考资料、搜集灵感等。但不鼓励借用新技术直接生成、批量制造低成本、低信息量内容。

**正面案例：** 该内容主要分享了通过提示词使用AI能力的过程，AI内容占比虽高，但同样包含个人原创信息而非照搬AI所生成的内容，融入了大量个人经验和思考，属于新技术辅助创作的正面案例。

**反面案例1（AIGC文字）：** 该内容采用了AI生成的原文，且未进行任何更改和调整，属于直接使用新技术生产的低价值AIGC内容。

**反面案例2（AIGC图片）：** 该内容以AIGC图片为主体，存在明显不合理之处且未提供增量信息，属于直接使用新技术生产的低价值AIGC内容。

## 二、诱导违规

> 来源：微信规则中心「行为规范」分类

诱导违规是微信平台高频处罚类型，指通过利益诱惑、胁迫、虚假承诺等方式诱导用户进行分享、关注、点击等操作。

### 2.1 诱导分享

**典型违规情形：**

**案例1（利诱分享）：** 文章末尾设置"分享到3个群聊，截图发送公众号即可获得XX资料包"，以虚拟奖品诱导用户分享至微信群或朋友圈。
- **违反条款**：运营规范 §3.3.1 诱导分享
- **处罚**：限制文章传播 + 降低账号推荐权重

**案例2（胁迫分享）：** 文章使用"不转不是中国人""转发后一生平安""请好心人转发一下"等情感胁迫语言，诱导用户分享。
- **违反条款**：运营规范 §3.3.1 诱导分享
- **处罚**：删文 + 警告，多次违规封禁群发功能

**案例3（活动诱导）：** 文章推广商家活动，要求"集满50个赞免费领取XX"，将公众号文章作为营销活动的传播工具。
- **违反条款**：运营规范 §3.3.1 诱导分享 + §6 营销推广
- **处罚**：删文 + 限制传播

### 2.2 诱导关注

**典型违规情形：**

**案例1（关注换资源）：** 文章提供部分教程内容后截断，要求"关注后回复'教程'获取完整版"，以内容作为筹码诱导关注。
- **违反条款**：运营规范 §3.3.2 诱导关注
- **处罚**：删文 + 警告

**案例2（关注换抽奖）：** 文章设置"关注本号并转发本文，即可参与抽取iPhone/红包"活动。
- **违反条款**：运营规范 §3.3.2 诱导关注 + §3.3.1 诱导分享
- **处罚**：删文 + 降权

### 2.3 诱导点击

**典型违规情形：**

**案例：** 文章使用"恭喜您中奖""您有一份快递待查收"等虚假信息作为标题或文中小程序卡片文案，诱骗用户点击以骗取广告收益。
- **违反条款**：运营规范 §4.21 诱骗点击
- **处罚**：删文 + 限制小程序卡片功能

## 三、违规导流

> 来源：微信规则中心「行为规范」分类

违规导流指通过文章、简介、评论等渠道，将用户引导至公众号之外的平台、页面或私人账号，是平台重点打击的行为。

### 3.1 内容导流

**典型违规情形：**

**案例1（二维码导流）：** 文章正文信息不完整，核心内容通过文末放置的二维码引导用户跳转至第三方网页、小程序或添加个人微信。
- **违反条款**：运营规范 §5.4 导流内容
- **处罚**：删文 + 限制外链功能

**案例2（链接堆砌）：** 文章内容空洞，主要篇幅为多个外链，每个链接引导至不同平台或商品页面，文章本身不具备独立阅读价值。
- **违反条款**：运营规范 §5.4 导流内容 + §7.3 信息量不足
- **处罚**：删文 + 降权

### 3.2 互动导流

**典型违规情形：**

**案例1（私人号导流）：** 文章正文或评论区中嵌入微信号（如"加V: xxxxx"），引导用户添加个人微信进行私域营销。
- **违反条款**：运营规范 §5.4 导流内容（互动导流）
- **处罚**：删文 + 警告，多次违规封禁评论功能

**案例2（社群导流）：** 文章末尾放置微信群二维码，引导"扫码进群获取更多内容"，将公众号用户导流至私域社群。
- **违反条款**：运营规范 §5.4 导流内容
- **处罚**：删文 + 降权

### 3.3 多重嵌套导流

**典型违规情形：**

**案例：** 文章引导用户关注公众号 → 自动回复推送小程序 → 小程序引导添加企业微信 → 企业微信引导付费购买。经过多重跳转，最终引导用户进行交易。
- **违反条款**：运营规范 §5.4 导流内容（多重嵌套导流）
- **处罚**：永久封禁相关功能 + 主体拉黑

## 四、过度营销

> 来源：微信规则中心「内容规范」分类

过度营销指文章内容以商业推广为主，缺乏实际信息价值，对用户造成骚扰。

### 4.1 纯广告内容

**典型违规情形：**

**案例1（产品广告）：** 文章全文为产品介绍，包含产品图片、功能描述、价格信息、购买链接/二维码，无独立评测或体验内容。
- **违反条款**：运营规范 §6 营销推广
- **处罚**：限制推荐 + 标记为广告

**案例2（促销信息）：** 文章以"限时特惠""清仓大甩卖"为主题，全文为折扣信息和购买引导。
- **违反条款**：运营规范 §6 营销推广
- **处罚**：删文 + 降权

### 4.2 软文式营销

**典型违规情形：**

**案例：** 文章以"测评""体验""分享"为标题，但正文中超过70%的篇幅在推销某一特定产品/服务，包含价格对比、购买渠道、促销码等信息，实质为广告。
- **违反条款**：运营规范 §6 营销推广 + §4.15 违反市场监管法律法规
- **处罚**：限制推荐 + 可能要求标注"广告"标识

### 4.3 保健品/健康产品过度营销

**典型违规情形：**

**案例：** 文章以"养生科普"为切入点，实际推销保健品，夸大产品功效（如"吃了XX保健品，三高全好了"），使用患者案例作为疗效证明。
- **违反条款**：运营规范 §5.1 不规范医疗科普 + §6 营销推广 + 广告法
- **处罚**：删文 + 可能涉及行政处罚

## 五、标题煽动、夸大、误导

> 来源：微信规则中心「内容规范」分类

标题问题是最常见的内容违规类型之一，平台对此类违规的处罚力度持续加大。

### 5.1 煽动情绪型标题

**典型违规情形：**

**案例1：** 标题使用"愤怒了！""气炸了！""全网震怒！"等极端情绪词，配合夸张描述，煽动读者情绪。
- **违反条款**：运营规范 §5.3.3 误导类标题
- **处罚**：限制推荐 + 删文

**案例2：** 标题以"紧急通知""速看""不看后悔一辈子"等制造紧迫感，但正文内容与标题严重程度不符。
- **违反条款**：运营规范 §5.3.3 误导类标题 + §4.21 诱骗点击
- **处罚**：限制推荐 + 降权

### 5.2 夸大型标题

**典型违规情形：**

**案例1：** 标题使用"史上最强""全球第一""绝无仅有"等极限词进行夸大描述。
- **违反条款**：运营规范 §5.3.3 + 广告法极限词
- **处罚**：删文 + 限制推荐

**案例2：** 标题使用未经证实的数据或比例，如"99%的人都不知道""10个人里9个都做过"，制造虚假的普遍性。
- **违反条款**：运营规范 §3 不实信息 + §5.3.3 误导类标题
- **处罚**：限制推荐 + 删文

### 5.3 误导型标题（标题党）

**典型违规情形：**

**案例1：** 标题暗示某重大事件发生（如"XX出事了！""XX被曝出惊天丑闻"），但正文内容与标题完全不一致，或关联内容篇幅极小。
- **违反条款**：运营规范 §5.3.3 误导类标题
- **处罚**：删文 + 降权

**案例2：** 标题故意隐藏关键信息制造悬念，如"他做了一件事，结果让所有人都惊了"，正文内容平淡无奇，标题与内容严重脱节。
- **违反条款**：运营规范 §5.3.4 关键信息缺失标题
- **处罚**：限制推荐

**案例3：** 标题冒充权威来源，如"央视曝光""官方紧急通知""新华社独家"，但内容来源不实。
- **违反条款**：运营规范 §3 不实信息 + §4.21 诱骗点击
- **处罚**：删文 + 严重警告

### 5.4 煽动群体对立型标题

**典型违规情形：**

**案例：** 标题使用"男人/女人就该……""XX地方的人都……""有钱人/穷人……"等群体标签化表述，煽动性别、地域、阶层对立。
- **违反条款**：运营规范 §4.25 煽动群体对立 + §4.2 传播不良价值观
- **处罚**：删文 + 降权

## 案例使用指南

在进行合规检查时，应参照本文件中的案例进行比对：

1. **定位违规类型**：根据文章内容和 DFA 扫描结果，确定可能涉及的违规类别
2. **查找对应案例**：在对应类别中找到最相似的官方案例
3. **引用案例**：在报告中引用案例编号和描述，如"类似问题参见本文件§2.1案例1（利诱分享）"
4. **评估严重程度**：参考案例中的处罚结果，评估当前文章的违规风险等级
5. **给出修改建议**：基于案例中的违规情形，给出针对性的修改方向
