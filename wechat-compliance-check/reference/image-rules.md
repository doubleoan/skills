# 图片合规规则

> 规则来源：微信公众平台图片内容审核规范、《网络信息内容生态治理规定》、广告法、平台实践

## 1. 封面图规范

### 尺寸要求
- **标准比例**：2.35:1（横图），推荐尺寸 900×383px 或等比例
- **最小分辨率**：不低于 900×500px（模糊图像会被降权）
- **文件大小**：单张不超过 5MB
- **格式**：JPG / PNG / GIF（GIF 封面只展示第一帧）

### 内容要求
- 封面图内容须与文章主题高度相关
- 禁止使用与正文无关的吸眼图片（如无关美女、夸张表情包）
- 封面图文字叠加不超过图片面积的 30%（参照广告审核标准）
- 禁止在封面图中显示联系方式（手机号、微信号、二维码）

### 高风险封面场景
| 场景 | 风险 |
|------|------|
| 涉及领导人/政府机构 logo | 极高 |
| 血腥、暴力、尸体照片 | 极高（直接删文） |
| 色情或擦边人物图 | 极高 |
| 儿童不雅图片 | 极高 |
| 假冒品牌 logo（如仿制微信、腾讯 logo） | 高 |
| 过度修图导致图片失真（容易被判低质） | 低-中 |

## 2. 正文配图规范

### 数量与格式
- 单篇文章图片数量无硬性限制，但过多图片影响加载体验
- 支持格式：JPG、PNG、GIF、WEBP
- 单张图片不超过 10MB

### 内容要求
- 图片须与该段落内容相关，不得使用无关配图误导读者
- 截图类配图须清晰可辨，关键信息不得遮挡
- 不得在正文图片中嵌入诱导关注/分享的引导语
- 不得在图片中显示其他平台的二维码（如抖音二维码）

### 截图使用注意
- 政策文件、法规截图：确认来源权威，注明出处
- 聊天记录截图：注意隐私，遮挡个人信息
- 媒体截图引用：标注原始媒体来源

## 3. 禁止的图片内容

### 绝对禁止（直接删文/封号风险）
- 色情、淫秽图片
- 血腥暴力、尸体、器官等令人不适的图片
- 涉及未成年人不雅内容
- 赌博、毒品相关图片
- 侮辱、歧视特定群体的图片
- 涉及政治敏感的图片（含领导人讽刺漫画等）

### 高风险（可能触发审核）
- 刀具、枪支特写图片
- 医疗手术过程血腥图片
- 含有完整身份证、银行卡号的图片
- 带有反动标语或标志的图片
- 大量密集文字的截图（影响审核识别）

## 4. 图片文字叠加规范

参照《互联网广告管理办法》：

- 图片内文字须清晰可读，不得有意模糊以规避审查
- 不得在图片中用文字承诺"最佳""第一""绝对"等极限表达
- 医疗健康类图片禁止使用"治愈""根治"等绝对化表述
- 禁止将广告伪装成新闻/政策通知格式
