> 规则来源：微信公众平台运营规范（https://mp.weixin.qq.com/mp/opshowpage?action=newoplaw）、微信规则中心违规示例（https://mp.weixin.qq.com/webpoc/ruleCenter）

## 1. 诱导行为（高频违规）

> 官方定义：通过利益诱惑、胁迫等方式诱导用户进行分享、关注、点击等操作。

### 1.1 诱导分享

通过外链或公众号消息等方式，强制或诱导用户将消息分享至朋友圈。

**具体违规情形：**
- **强制分享**：分享后才能继续下一步操作（如"分享后方可订阅""分享后方可继续看视频"）
- **利诱分享**：分享后对用户有奖励（如"邀请好友助力""集赞可获得抽奖机会"）
- **胁迫/煽动分享**：用夸张言语胁迫、引诱用户分享（如"不转不是中国人""请好心人转发一下""转发后一生平安""转疯了""必转"）

**处罚**：首次违规限制文章传播，多次违规降权、封禁群发功能，严重者永久封号。

**检测关键词/模式：**
```
分享到朋友圈、转发本文、转发即可、集赞、分享后、转发后
不转不是、请转发、必转、转疯了、转发可领取、分享给好友
```

### 1.2 诱导关注

通过奖励或其他方式，强制或诱导用户关注公众号。

**具体违规情形：**
- **强制关注**：用户需要关注才能获得好处或进行下一步操作
- **利诱关注**：关注后对用户有奖励（如"关注可获得红包""关注可参与抽奖"）
- **暗示关注**：以"关注获取更多""关注不迷路"等暗示性表述引导关注

**处罚**：首次警告并删文，多次违规限制推荐、降权。

**检测关键词/模式：**
```
关注后回复、关注即可、关注获取、关注领取、关注参与
扫码关注、关注不迷路、关注后可查看、关注后解锁
```

### 1.3 诱导点击

以虚假信息或夸大描述引诱用户点击文章、链接或小程序卡片。

**具体违规情形：**
- 文章使用不完全/擦边的标题和封面，插入诱导性小程序卡片，诱骗用户点击骗取广告收益
- 以"点击阅读全文即可领取"等方式诱导点击
- 使用"恭喜您中奖"等虚假信息诱骗用户点击链接

**检测关键词/模式：**
```
点击阅读全文即可、恭喜您中奖、点击领取、点击查看
您有一份、立即查看、马上领取、点击这里
```

## 2. 违规导流（高频违规）

> 官方定义：通过文章、简介、评论等渠道，将用户引导至公众号之外的平台、页面或私人账号。

### 2.1 内容导流

发布不完整内容，并通过二维码、文字、链接、图标等形式进行导流，主要目的是将用户导至其他内容、小程序、其他公众号、视频号、第三方网页或应用。

**具体违规情形：**
- 文章正文故意截断，引导用户"点击阅读原文"或扫码跳转
- 文中嵌入大量外链，文章本身信息量不足，主要目的是导流
- 文章以封建迷信、违规擦边标题吸引点击，主要目的是诱导跳转到其他页面

**检测关键词/模式：**
```
点击阅读原文、长按二维码、扫码查看、扫码获取
复制链接打开、点击阅读原文查看、更多详情请见
点击下方链接、点击这里查看
```

### 2.2 互动导流

通过账号简介、内容、评论等途径，直接或间接嵌入个人联系方式、外部链接、二维码等引导信息。

**具体违规情形：**
- 文中嵌入微信号、手机号、QQ号等联系方式引导添加好友
- 文中放置二维码引导添加私人账号或进入微信群
- 文中嵌入外部链接引导至第三方平台

**检测关键词/模式：**
```
加微信、加V、添加微信、微信号、WX:、VX:
QQ群、QQ号、加群
长按二维码加、扫码进群、扫码加
```

### 2.3 多重嵌套导流

通过定向引流、多重跳转后，最终引导用户进行线上或线下交易，或发生可能骚扰用户、损害其交易安全与合法权益的行为。

**检测关键词/模式：**
```
长按识别、扫码进入、点击跳转、查看更多详情
点击下方、识别图中二维码、跳转到、打开链接
```

> 多重嵌套导流通常不是单一关键词可检测的，需结合上下文判断：文章 → 自动回复 → 小程序 → 添加微信 → 付费购买的完整链路。

## 3. 功能滥用

### 3.1 滥用原创声明

如下情形不得声明原创：
- 整合引用内容：未经授权不合理引用或大篇幅整合网络信息、公共信息、他人作品
- 公共公开内容：法律法规、政府文件、时事新闻、天气预报等
- 营销宣传内容：广告宣传、商家活动、招聘、投票征稿
- 非独家授权内容：获得授权但非独家
- 无意义内容：胡诌文字、大段堆砌、乱凑字数
- 蹭原创内容：文字简单无实际意义，主要篇幅为搬运或导流

> **AI 辅助创作规则**：AI 辅助创作的文章可以声明原创（平台正面案例已确认），但必须融入大量个人经验和思考。直接贴 AI 生成原文不做加工不可声明原创。

### 3.2 滥用赞赏功能

- 禁止将赞赏功能用于募捐、赌博或售卖商品
- 不得通过赞赏功能进行变相收费

### 3.3 滥用模板消息/客服接口

- 不得通过模板消息、客服接口发送骚扰或营销信息
- 不得滥用付费功能和订阅通知

### 3.4 恶意编辑

- 文章内容不能过于简短空洞（正文极少）
- 不得通过隐藏内容谋取广告费
- 不得使用特殊代码、排版、文字刷白、特殊背景色等方式恶意规避系统检测

**检测要点（功能滥用）：**
- 文章正文是否过于简短（少于 200 字且无实质内容）→ 可能触发恶意编辑
- 文章是否声明原创但内容以引用/搬运为主 → 可能滥用原创声明
- 文中是否出现赞赏引导用于非创作目的 → 可能滥用赞赏

## 4. 违规经营与资质缺失

> 官方规定：无资质从事以下特许业务属违规行为。

### 需要特定资质的领域

| 领域 | 需要的资质 | 文章风险点 |
|------|-----------|-----------|
| 金融/理财 | 金融牌照 | 推荐具体理财产品、给出投资建议 |
| 医疗健康 | 医疗执业许可 | 提供诊疗建议、推荐药品/疗法 |
| 烟草 | 烟草专卖许可 | 任何形式的烟草推广 |
| 虚拟货币/数字藏品 | 明确禁止 | ICO推广、NFT交易引导 |
| 野生动植物 | 特许许可 | 珍稀动植物交易信息 |
| 婚恋交友 | 相关许可 | 无资质提供婚恋服务 |
| 荐彩 | 明确禁止 | 任何形式的彩票推荐 |

### 检测要点
- 文章涉及金融建议时，是否声明"不构成投资建议""投资有风险"
- 文章涉及健康建议时，是否声明"不替代专业医疗诊断"
- 文章是否推广了需要资质但作者不具备的服务

## 5. 欺诈与多级分销

### 5.1 多级分销（类传销）
- 以拉人头、发展下线为主要盈利模式
- 入门费 + 层级提成 + 团队奖励的三级以上分销结构

### 5.2 网赚骗局
- 承诺"日赚千元""躺赚"等不实收益
- 以"兼职刷单""在家创业"为名骗取信息或金钱

### 5.3 虚假刷单与有偿投票
- 组织或推广刷单行为
- 提供有偿投票服务

**检测关键词/模式：**
```
日赚、躺赚、零成本创业、在家赚钱、刷单、有偿投票
发展下线、团队奖励、入门费、层级提成
```

## 6. 恶意对抗与规避检测

以下行为被视为恶意对抗，将受到严厉处罚（永久封禁 + 主体拉黑）：

- **批量注册**：注册多个相似账号进行违规操作
- **盗用身份**：冒用他人身份信息注册或运营
- **多次违规不改**：被处罚后继续违规
- **绕过审核**：使用技术手段规避平台检测（如特殊字符、隐藏文字、图片替代文字等）
- **使用第三方违规工具**：刷粉软件、群发突破限制工具等

> **与文章的关系**：文中不应出现引导用户使用上述手段的内容。

## 7. 强制关注

> 2023年微信重点整治方向

- **扫码点餐强制关注**：用户扫码点餐时必须关注公众号，无跳过选项
- **扫码连WiFi强制关注**：连接WiFi需要先关注公众号
- **其他场景**：任何需要用户关注公众号才能获取基本服务的场景

> **与文章的关系**：如果文章推广的服务涉及扫码场景，需确认不存在强制关注行为。

## 8. AI 自动化与模板化发布

> 2024-2025年微信新增规则

### 8.1 非真人 AI 自动化批量发文
- 使用 AI 完全替代人工，批量生成并发布大量文章
- 文章无任何个人经验、观点或原创内容，纯 AI 产出
- 同一账号短时间内发布大量结构相似、主题雷同的 AI 文章

### 8.2 套路化模板写作
- 使用固定模板 + 关键词替换的方式批量生成文章
- 文章结构完全一致，仅替换部分变量（如地名、产品名）
- 同一模板被多个账号使用

### 8.3 AIGC 标识要求
- 根据2025年微信新规，AI 生成内容必须显著标识
- 未标识的 AI 生成内容，平台可能限制推荐或标记为低质量
- AI 辅助创作（人工深度参与）与 AI 直接生成（人工参与少）有本质区别

**检测要点：**
- 文章是否声明了 AI 辅助创作（如果 AI 内容占比高）
- 文章是否体现了个人经验和思考（区分辅助 vs 直接生成）
- 与近期发布的文章是否结构高度雷同（模板化检测）

## 9. 行为违规关键词与模式汇总

以下为检测时需要关注的行为层面风险关键词和模式：

### 高风险模式
| 模式 | 关键词示例 | 对应违规类型 |
|------|-----------|-------------|
| 利益诱导分享 | 转发领取、分享得红包、集赞 | 诱导分享（§1.1） |
| 胁迫分享 | 不转不是、请转发、必转 | 诱导分享（§1.1） |
| 关注换取 | 关注后回复、关注获取、关注领取 | 诱导关注（§1.2） |
| 外部导流 | 加微信、扫码进群、点击阅读原文 | 违规导流（§2） |
| 联系方式暴露 | 微信号:、VX:、QQ群 | 互动导流（§2.2） |
| 虚假收益承诺 | 日赚千元、躺赚、零成本 | 欺诈（§5.2） |
| 无资质服务 | 推荐理财、推荐药品、荐股 | 资质缺失（§4） |
| AIGC 无标识 | （需整体评估，非关键词匹配） | AI 滥用（§8） |
