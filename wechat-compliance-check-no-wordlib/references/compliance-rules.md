# 政治敏感词+广告法违禁词+合规替代速查

> 本文件为 platform-rules.md 和 platform-rules-basic.md 的补充，覆盖两者未详述的违禁词库和替代方案。

---

## 一、政治敏感词检测规则

### 1.1 绝对禁区（直接封号级）

| 类别 | 检测规则 |
|------|---------|
| 国家领导人 | 不得出现对领导人的不敬/调侃/戏谑；不得伪造言论；不得假借名义营销 |
| 国家机关 | 不得假借国家机关名义发布信息或营销；不得使用"国家机关推荐""专供政府" |
| 国家象征 | 不得商业化使用国旗/国徽/党旗/军徽元素 |
| 分裂国家 | 不得出现台独/港独/藏独/疆独等分裂国家主权表述 |
| 政治事件 | 不得对政治事件歪曲解读/煽动性评论 |
| 涉军内容 | 不得军事泄密/丑化军队形象 |

### 1.2 技术类文章特检

| 场景 | 检测规则 |
|------|---------|
| 外企/外国技术报道 | 不得使用"境外势力""文化入侵"等极端政治化表述 |
| AI伦理讨论 | 避免与政治体制做不当类比 |
| 数据安全讨论 | 不得泄露国家信息安全相关敏感信息 |
| 开源项目评论 | 不得将技术选择政治化（如"用国产XX才是爱国"） |

## 二、广告法违禁词库

### 2.1 绝对化用语（技术类最高频违规）

**"最"字系列**：最、最佳、最具、最优、最好、最大、最高、最高级、最先进、最便宜、最受欢迎
**"一"字系列**：第一、唯一、NO.1、TOP.1、独一无二、全国第一、一流
**"级/极"系列**：国家级、顶级、极品、极致、终极、极佳
**"首/家/国"系列**：首个、首选、独家、首发、全国销量冠军、中国驰名

### 2.2 虚假误导类

| 类别 | 违禁表述 | 合规替代 |
|------|---------|---------|
| 效果保证 | 包治百病、立竿见影、永不反弹、根治 | 辅助改善、实际效果因人而异 |
| 虚假承诺 | 史无前例、前无古人、永久、万能、100% | 长久、多用途、提升百分之X |
| 欺诈诱导 | 点击领奖、恭喜获奖、全民免单 | 直接删除 |
| 诱导消费 | 秒杀、抢爆、万人疯抢、卖疯了 | 限时特惠至X月X日 |

### 2.3 对比贬低类

| 违禁表述 | 合规替代 |
|---------|---------|
| 比XX品牌好、吊打同行、XX克星 | 与自身产品对比（"升级款提升XX"） |
| 超越所有同类 | 贴合自身优势（"适配XX场景"） |

### 2.4 行业专属禁词

**医疗健康**：治愈率99%、根治、安全无毒副作用 → 遵医嘱使用、辅助缓解XX症状
**金融理财**：保本保收益、零风险高回报、稳赚不赔 → 投资有风险、过往业绩仅供参考
**教育培训**：保过、100%录取、零基础过雅思 → 助力提升、往期通过率XX（附真实数据）
**食品保健**：抗癌、降三高、治疗糖尿病 → 辅助调理、补充XX营养素

## 三、合规替代速查表

| 违禁词 | 合规替代 | 违禁词 | 合规替代 |
|--------|---------|--------|---------|
| 第一、唯一 | 首要、前列、优选 | 完美 | 精致、出色 |
| 最好、最佳 | 优质、高效、出众 | 万能 | 全能、多用途 |
| 最先进 | 先进、前沿 | 权威 | 专业、资深 |
| 顶级、极致 | 高端、精良、出色 | 遥遥领先 | 表现突出、领先梯队 |
| 独一无二 | 少有、特色鲜明 | 碾压、吊打 | 在XX维度上表现优异 |
| 永久 | 长久、持续 | 秒杀 | 领先于 |
| 100% | 提升百分之X | 史无前例 | 鲜有先例 |
| 绝对、彻底 | 有效、显著 | 全球最快 | 测试环境下XX指标达Y |

## 四、处罚等级参考

| 违规等级 | 典型违规 | 平台处罚 |
